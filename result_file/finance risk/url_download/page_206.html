This template includes collapsible lists.

    • To set it to display all lists when it appears (i.e. all lists expanded), use:
    •  {{Finance sidebar |expanded=all}} or, if enabled, {{Finance sidebar |all}}  (i.e. omitting "expanded=").
    • To set it to display one particular list while keeping the remainder collapsed (i.e. hidden apart from their headings), use:
    •  {{Finance sidebar |expanded=listname}} or, if enabled, {{Finance sidebar |listname}}
    • …where listname is one of the following (do not include any quotemarks):
      • markets
      • instruments
      • corporate
      • personal
      • public
      • banking
      • reg/stands
      • history
    • For example, {{Finance sidebar |expanded=reg/stands}} or, if enabled, {{Finance sidebar |reg/stands}}
